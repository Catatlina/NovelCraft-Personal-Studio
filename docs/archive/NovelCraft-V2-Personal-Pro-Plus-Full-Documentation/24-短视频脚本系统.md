# 短视频脚本系统

## 概述

将文字内容转化为短视频脚本，覆盖抖音/小红书/B站三大平台。

## 脚本结构

每种平台脚本包含完整字段：

| 字段 | 说明 |
|---|---|
| 3 秒钩子 | 开篇吸引注意力的金句/问题/冲突 |
| 分镜 | 场景划分，每镜 3~15 秒 |
| 旁白 | 画外音解说 |
| 口播稿 | 出镜台词 |
| 镜头建议 | 景别/运镜/画面描述 |
| 标题 | 视频标题（含 emoji 和关键词） |
| 封面文案 | 封面文字建议 |
| 评论引导 | 引导互动的话术 |

## 平台差异

| 平台 | 时长 | 风格 | 特点 |
|---|---|---|---|
| 抖音 | 15~60s | 快节奏/强反转/口语化 | 前 3 秒定生死 |
| 小红书视频 | 30~120s | 精致/生活方式/清单体 | 封面 + 标题关键 |
| B站 | 2~10min | 深度/弹幕互动/中二 | 允许更长叙事 |

## 输入源

- 小说章节 → 场景化脚本
- 自媒体文章 → 口播稿
- 知识内容 → 科普脚本
- 产品介绍 → 种草脚本

## AI 生成能力

- 自动识别输入内容的高光点 → 转化为钩子
- 文字描述 → 分镜画面建议
- 自动适配平台风格（抖音快/B站长/小红书精）
- 批量生成（一篇长文 → 抖音×3 + 小红书×2 + B站×1）
