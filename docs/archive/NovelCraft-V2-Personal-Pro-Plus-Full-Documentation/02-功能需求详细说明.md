# 功能需求详细说明

> 本文档逐一展开 13 大类功能的具体需求、交互流程和验收标准。

## 1. 小说创作系统

### 1.1 创作流水线
灵感输入（一句话到一段） → 市场分析（可选）→ 书名生成（3 候选）→ 简介与卖点 → 世界观构建 → 人物设定（3~6 个）→ 总纲 → 分卷大纲 → 章节细纲 → 正文 → 审核 → 发布

### 1.2 防崩体系
- **7 层上下文装配**：全书状态摘要/本卷摘要/近3章摘要/实体状态表/到期伏笔/素材RAG/本章细纲
- **实体状态表**：逐章更新人物位置/关系/持有物/已知信息
- **伏笔系统**：埋设→追踪→回收，到期强制提醒
- **检测族**：OOC/设定冲突/前文一致性/节奏检测
- **防崩策略**：章纲先审→交叉模型审→每10章巡检→卷级复盘门禁

### 1.3 AI 操作集
- 选中续写/改写/润色/扩写/缩写/去AI味
- 右键 AI 菜单
- ⌘K 全局命令

## 2. 自媒体创作系统

### 2.1 支持平台
国内：微信公众号/今日头条/小红书/百家号/知乎/大鱼号/网易号
海外：Medium/Substack/X(Twitter)

### 2.2 一稿多平台
同一篇源内容 → fan-out 生成各平台适配版本：
- 公众号：长文 + 排版 + 引导关注
- 头条：SEO 标题 + 关键词 + 推荐优化
- 小红书：短笔记 + emoji + 标签 + 封面建议
- 知乎：问答格式 + 深度分析
- Medium/Substack：英文长文 + SEO

## 3. 热点监控与选题系统

### 3.1 数据源（可插拔 adapter）
- 新闻热榜（微博/百度/知乎热榜）
- 行业热点（AI/科技/财经/商业）
- 平台榜单（起点/番茄/小红书/抖音）

### 3.2 分析能力
热点价值评估 → 平台适配判断 → 选题推荐 → 选题库管理
趋势分析/爆款分析/扫榜/传播潜力分析

### 3.3 每日晨报
定时抓取热点 → AI 分析 → 生成选题建议 → 生成多平台草稿 → 等待审核

## 4. Knowledge Hub

### 4.1 统一入库
文本/图片/PDF/Word/Markdown/网页链接/小说原文 → 自动解析 → 向量化 → 入库

### 4.2 四库
- 金句库：精彩句子/对白/描写
- 标题库：爆款标题模板
- 平台规则库：各平台内容规范
- 品牌风格库：创作者风格卡

### 4.3 风格学习
样本→特征提取→style_card（叙事人称/句长分布/高频意象/节奏特征）→注入生成上下文

### 4.4 防侵权红线
相似度 ≥ 0.75 强制重写；0.6~0.75 人工确认；< 0.6 放行

## 5. 编辑器

- Tiptap 富文本编辑器（块编辑/斜杠命令/Markdown 快捷输入）
- 三栏工作台：目录/人物/设定 | 正文 | AI 助手
- 自动保存 + 版本快照
- 暗色模式原生支持
- 选中浮条：续写/改写/润色/扩缩写
