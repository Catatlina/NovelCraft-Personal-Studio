# 自媒体生产系统

## 概述

覆盖 10 个主流内容平台的专业化自媒体内容生产工具。

## 支持平台

| 平台 | 内容类型 | 特色能力 |
|---|---|---|
| 微信公众号 | 长文/图文 | 排版美化/引导关注/话题标签 |
| 今日头条 | 文章/微头条 | SEO 标题/关键词优化/推荐算法适配 |
| 小红书 | 笔记 | emoji 风格/标签系统/封面建议 |
| 百家号 | 文章 | 百度 SEO/关键词密度 |
| 知乎 | 问答/文章 | 深度分析/专业引用/盐值优化 |
| 大鱼号 | 文章 | UC 推荐适配 |
| 网易号 | 文章 | 跟帖引导 |
| Medium | Long-form | 英文 SEO/Clap 优化 |
| Substack | Newsletter | 邮件标题/订阅引导 |
| X (Twitter) | Thread | 线程拆分/hook 开头 |

## 平台风格适配

每个平台有独立的风格 Profile：
- **语气**：正式/口语化/专业/亲切
- **长度**：平台最优字数区间
- **结构**：标题-导语-正文-CTA 布局
- **标签**：平台最优标签数量和类型
- **排版**：分段/emoji/引用/列表偏好

## 一稿多平台 Fan-out

核心价值：一篇源内容 → 一键生成 10 个平台适配版

```
源内容（如公众号长文）
  ├─ 头条版：SEO 标题 + 关键词 + 微头条摘要
  ├─ 小红书版：短笔记 + 5~8 emoji + 10 标签 + 封面文案
  ├─ 知乎版：问答格式 + 深度分析 + 引用
  ├─ 百家号版：百度 SEO + 关键词密度优化
  ├─ Medium版：英文翻译 + 文化适配
  └─ 更多...
```

## 内容字段族

每种平台类型自动生成完整的内容字段：
- 爆款标题（3 候选）
- SEO 标题
- 摘要/导语
- 标签/关键词
- 封面文案
- 配图建议
- 评论区引导
- 行动号召（CTA）

## 批量生产

- 批量生成（同主题 × 5 平台）
- 内容矩阵管理页（列表视图/看板视图）
- 定时生产（配合热点晨报）
