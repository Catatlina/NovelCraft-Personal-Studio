# 短篇小说生产系统

## 概述

短篇/微小说/情感故事/悬疑故事/爆款短篇的专业化生产工具。

## 支持的短篇类型

| 类型 | 字数范围 | 典型平台 |
|---|---|---|
| 微小说 | 500~2000 字 | 小红书/微博/知乎 |
| 短篇故事 | 2000~10000 字 | 公众号/头条/知乎 |
| 情感故事 | 1000~5000 字 | 小红书/公众号 |
| 悬疑短篇 | 3000~15000 字 | 知乎/头条 |
| 爆款短篇 | 2000~8000 字 | 各平台通用 |

## 爆款结构模板

系统内置多种经市场验证的短篇结构模板：
- **反转式**：铺垫 → 误导 → 反转
- **钩子式**：爆点开篇 → 回溯 → 高潮
- **情感式**：日常 → 冲突 → 和解/遗憾
- **悬疑式**：谜题 → 线索 → 真相
- **对话式**：以对话推动剧情

## 生产流程

```
灵感/选题 → 选择类型与模板 → AI 生成全文 → 审核 → 编辑润色 → 发布
```

相比长篇，短篇流程更短：
- 跳过世界观/人物设定（内联在生成中）
- 全文一次生成（非逐章）
- 快速审核（7 维 → 精简为 4 维：文笔/剧情/情感/爆款潜力）

## AI 操作

- 一键生成全文
- 选中改写/润色/扩缩写
- 结尾调整（反转/留白/升华）
- 去 AI 味优化
- 多平台格式派生（同一短篇 → 公众号版/小红书版/知乎版）

## 知识复用

- 风格学习：分析爆款短篇 → 提取节奏/结构/句式特征 → 应用于新生成
- 金句库：短篇高光句子自动入库，后续生成时注入
