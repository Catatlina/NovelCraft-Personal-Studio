# UI 设计规范

## 设计原则

**UI 第一** — 对标 Cursor / Notion / Linear / Raycast / ChatGPT

- **Calm by default**：默认安静，低饱和中性底，高亮只给"此刻重要的事"
- **Dense but breathe**：高密度 + 规律留白
- **Keyboard is first-class**：每个动作有快捷键，可被 ⌘K 触发
- **Dark-native**：暗色为原生默认，亮色是变体
- **Motion with restraint**：动效只在"状态变化有意义"处出现，≤200ms

## 对标基准

| 维度 | 对标产品 | 学什么 |
|---|---|---|
| 信息密度 | Linear | 列表紧凑、行高克制、状态用色点、键盘优先 |
| 编辑沉浸 | Notion / Cursor | 块编辑流畅、选中即出 AI 浮条、无模式切换 |
| 命令交互 | Raycast | 全局 ⌘K、模糊搜索、动作即命令 |

## 色彩系统

### 品牌色（低饱和靛蓝 — 写作/思考的冷静感）
- `--brand-500`: #5B66DB (dark) / #4F5BD6 (light) — 主色/主按钮
- `--brand-600`: #444FCB — 按压态

### 中性色（暗色偏冷灰，亮色暖中性）
- `--bg-base`: #14161C (dark) / #FFFFFF (light)
- `--bg-subtle`: #1C1F26 (dark) / #F6F7F9 (light) — 卡片
- `--text-primary`: #E8EAF0 (dark) / #1E2230 (light)
- `--text-secondary`: #A6ABBA (dark) / #595F70 (light)

### 语义色
- `--success`: #31B572 — 通过/成功
- `--warning`: #F2A93B — 风险/警告
- `--danger`: #DE4B5E — 错误/拦截

## 排版

- 字体：系统原生字体栈（-apple-system, Inter, sans-serif）
- 编辑器：等宽可选（JetBrains Mono / SF Mono）
- 字号阶梯：12/13/14/15/16/18/20/24/30/36
- 行高：1.5（正文）/ 1.3（标题）/ 1.2（密集列表）

## 间距系统（4px 基准）

`--space-1`: 4px | `--space-2`: 8px | `--space-3`: 12px | `--space-4`: 16px | `--space-6`: 24px | `--space-8`: 32px | `--space-12`: 48px

## 圆角

`--radius-sm`: 4px | `--radius-md`: 8px | `--radius-lg`: 12px | `--radius-full`: 9999px

## 阴影（暗色用发光感，亮色用投影）

```css
--shadow-sm: 0 1px 2px rgba(0,0,0,0.3);
--shadow-md: 0 4px 12px rgba(0,0,0,0.4);
--shadow-lg: 0 8px 24px rgba(0,0,0,0.5);
```

## 核心组件

Button / Input / Select / Dialog / Toast / Tooltip / Popover / DropdownMenu / Tabs / Progress / Badge / Card / Table / EmptyState / Skeleton / Command(⌘K) / RadarChart / DiffView

均基于 Radix UI + shadcn/ui 定制，统一通过 Design Tokens 控制视觉。
