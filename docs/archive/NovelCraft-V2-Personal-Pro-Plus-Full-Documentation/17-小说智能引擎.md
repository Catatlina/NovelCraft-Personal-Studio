# 小说智能引擎

## 概述

小说智能引擎是 C6 叙事一致性的完整实现，覆盖从大纲到正文的完整创作链路。

## 创作流水线

```
灵感 → 市场分析 → 书名生成 → 简介卖点 → 世界观 → 人物设定
  → 百万字总纲 → 分卷大纲 → 章节细纲 → 正文生成 → 审核 → 发布
```

## 核心子系统

### 1. 大纲生成系统
- **总纲**：全书结构/核心冲突/主线支线/关键转折点
- **卷纲**：分卷目标/人物弧线阶段/伏笔计划
- **细纲**：章节级写作指导（场景/冲突/人物出场/信息揭露）

### 2. 正文生成系统
- 7 层上下文装配（见 Context Hub）
- 按细纲 + 上下文逐章生成
- 自动续写：上一章结尾 → 下一章开头无缝衔接

### 3. 防崩体系
- **章纲先审**：大纲比正文便宜 20 倍，先审大纲再生成
- **交叉模型审**：生成用 DeepSeek，审核用 Claude，不同模型交叉把关
- **每 10 章巡检**：全局一致性扫描
- **卷级复盘门禁**：每卷完成后全面审查

### 4. 伏笔系统
- 种植：写手在正文中标记伏笔（或 AI 自动识别）
- 追踪：伏笔状态看板（已埋/部分揭示/已回收）
- 回收：到期强制注入上下文 + Reviewer 校验

### 5. 质量审核（7 维）
| 维度 | 检测内容 |
|---|---|
| 文笔 | 语言流畅度/描写质量 |
| 剧情 | 节奏/冲突/悬念 |
| 人物 | OOC/弧线偏移 |
| 设定 | 世界观一致性/规则冲突 |
| 逻辑 | 前文一致性/时间线矛盾 |
| 伏笔 | 回收状态/遗忘检测 |
| 商业 | 爽点密度/读者吸引力 |

## 编辑器操作集

选中即可调用的 AI 操作：
- **续写**：从选中处继续生成
- **改写**：换表达方式不换意思
- **润色**：优化文笔/增强画面感
- **扩写**：扩充细节/场景/心理描写
- **缩写**：精简冗余
- **去 AI 味**：消除模板化表达，增加人味
