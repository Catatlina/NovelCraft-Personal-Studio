# 前端工程规范

## 目录结构

```
frontend/src/
├── design/                # tokens.css(明暗双套) · primitives(Radix封装) · motion.ts
├── components/
│   ├── editor/            # Tiptap扩展：AI浮条/行内批注/版本树面板/diff视图
│   ├── workflow/          # RunTimeline · NodeCard · HumanGate · DAG编辑器
│   ├── novel/             # CharacterCard · OutlineTree · RadarScore · ForeshadowBoard
│   ├── studio/            # ContentMatrix · KnowledgeBrowser · TrendBoard
│   └── common/            # CommandK(⌘K) · DataTable · EmptyState
├── pages/                 # 按里程碑拆分
├── stores/                # zustand: auth/project/run(SSE)/editor/offline
├── offline/               # sw.ts(ServiceWorker) · db.ts(IndexedDB) · outbox.ts
├── api/                   # openapi-ts 生成客户端
└── hooks/                 # useSSE useAutosave useOffline useBudget useCommandK
```

## 技术选型

| 层 | 选型 |
|---|---|
| 框架 | React 18 + TypeScript strict |
| 构建 | Vite |
| UI | Radix UI + shadcn/ui + Tailwind CSS |
| 编辑器 | Tiptap (ProseMirror) |
| 状态 | Zustand (客户端) + TanStack Query (服务端) |
| 离线 | vite-plugin-pwa + IndexedDB |
| 动效 | Framer Motion（克制使用，≤200ms） |
| API | openapi-ts 生成类型安全客户端 |

## 设计系统（UI 第一）

所有视觉决策来自 Design Tokens，**禁止在业务代码写裸色值/裸间距/裸字号**：

```css
/* ✅ 正确 */
.button { background: var(--brand-500); padding: var(--space-3); }

/* ❌ 错误 */
.button { background: #5B66DB; padding: 12px; }
```

## 暗色模式

- `[data-theme="dark"]` 为默认
- 所有组件必须支持亮暗双套
- 切换通过 CSS 变量级联，不触发重渲染

## 组件规范

- 每个组件有 Storybook story
- Props 必须有 TypeScript 接口
- 空状态/加载态/错误态/边界情况全部处理
- 优先用 Radix 无头组件 + Tailwind 样式，不重复造轮子

## 性能

- 路由级代码分割（React.lazy）
- 大列表虚拟滚动
- 编辑器内容增量同步（debounce 1s 自动保存）
- SSE 连接复用

## Git 规范

- ESLint + Prettier（保存自动格式化）
- TypeScript strict mode（无 any）
- PR 必须通过 lint + type-check + build
