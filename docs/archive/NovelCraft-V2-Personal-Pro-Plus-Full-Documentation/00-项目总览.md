# NovelCraft V2.0 Personal Pro+ 项目总览

> 版本：V2.0 ｜ 定位：个人 + 1~5 人小团队的 AI 内容创作与运营工作台

## 产品定义

**NovelCraft Personal Studio — AI Content Studio for Writers & Creators**

一个人，也能拥有专业内容团队的生产力。

## 核心定位

不是"一个人运营 AI 内容公司的操作系统"，而是"一个人或小团队（1~5 人）的 AI 内容创作与运营工作台"。这一定位确保：
- 不做过度复杂的企业架构
- 保留全部生产能力
- 后续可扩展为 SaaS

## 完整闭环

选题 → 创作 → 审核 → 发布 → 数据反馈 → 持续优化

## 10 大产品原则

1. **功能完整，不做阉割** — 小说/自媒体/热点/内容复用/短视频/AI/工作流/知识/发布/出海，全部保留
2. **架构轻量** — React + FastAPI + PostgreSQL + Redis + Docker Compose，单 VPS
3. **UI 第一** — 对标 Cursor/Notion/Linear/Raycast
4. **AI 无处不在** — 不是聊天窗口，是右键/选中/内联/⌘K
5. **创作体验第一** — 流畅流程，不是 100 个按钮
6. **Everything is Content** — 统一内容模型，小说=公众号=头条=小红书=视频脚本
7. **Everything is Knowledge** — 人物/世界观/热点/文章/Prompt 统一入库
8. **Everything is Traceable** — AI 调用全记录，输入/输出/Prompt/模型/Token/耗时
9. **Everything is Versioned** — 正文/Prompt/人物/世界观/知识全部版本化
10. **Automation First** — 所有重复工作自动化

## 明确不做

❌ 微服务 ❌ Kubernetes ❌ SaaS 多租户 ❌ 企业组织架构 ❌ 支付系统 ❌ 复杂权限树

## 技术栈

React + Vite + TS → FastAPI → PostgreSQL + pgvector → Redis → Celery → Docker Compose
