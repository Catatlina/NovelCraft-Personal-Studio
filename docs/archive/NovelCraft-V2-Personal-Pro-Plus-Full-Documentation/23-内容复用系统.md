# 内容复用系统

## 概述

最大化内容价值——一篇素材，多种形态，多平台分发。

## 复用形态

| 源形态 | 目标形态 | 说明 |
|---|---|---|
| 小说章节 | 公众号故事 | 提取高潮片段/情感段落 |
| 小说设定 | 小红书世界观科普 | 图文化/清单化 |
| 长文 | 知乎回答 | 问题化重组 |
| 长文 | 短视频脚本 | 场景化/对白化 |
| 图文 | 纯文本 | 去格式化 |
| 分析文章 | Twitter Thread | 要点拆分 |

## 复用工作流

```
选择源内容 → 选择目标平台/形态 → AI 分析源内容 → 生成适配版
  → 审核 → 发布
```

支持：
- **单篇复用**：一篇 → 一个目标
- **批量复用**：一篇 → 多目标（fan-out）
- **定时复用**：配合工作流定时执行

## 内容去重与变异

同一素材多次复用时，AI 确保：
- 表达不重复（不同角度/不同案例/不同结构）
- 适配平台差异（不是简单替换标题）
- 保持品牌风格一致

## 衍生内容追溯

- 每篇衍生内容记录源内容
- 数据回流聚合到源内容
- "一篇公众号文章火了" → 查看它衍生了哪些平台版本 → 各平台表现如何
