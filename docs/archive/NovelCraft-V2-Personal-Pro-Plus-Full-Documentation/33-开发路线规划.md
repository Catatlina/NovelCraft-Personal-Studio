# 开发路线规划

## 里程碑总览

全量交付约 9~10 个月，严格按里程碑制：M(n) 未验收不开 M(n+1)。

| 里程碑 | 周期 | 一句话交付 |
|---|---|---|
| **M1 地基+MVP** | 第 1~5 周 | 灵感→第一章闭环，平台骨架定型 |
| **M2 小说引擎** | 第 6~16 周 | 30 万字防崩连载 + 工作流可视化 |
| **M3 内容工作室** | 第 17~26 周 | 全体裁生产 + 知识库 + 热点晨报 |
| **M4 发布数据出海** | 第 27~34 周 | 生产→发布→回流→ROI 闭环 |
| **M5 协作离线多端** | 第 35~40 周 | 小团队 + 离线 + 移动 |

## M1 地基+MVP（P0）

**目标**：设计系统建成；C1/C2/C3/C5/C8 五个平台能力以最终表结构落地；跑通"注册→灵感→六件套→审核→编辑→保存"。

**核心任务**：
- V1 遗留收口（106 测试全绿）
- 24 表骨架一次性建齐 + V1 数据迁移
- AI Gateway v1（DeepSeek + 追踪 + 三级预算）
- 工作流引擎 v0（chain+human 节点）
- bootstrap 预设工作流（8 节点全链）
- 设计系统 v1（tokens + 12+ 组件）
- 前端五页（向导/进度/审阅/编辑器/成本）
- SSE 进度推送

## M2 小说引擎完全体（P1）

**目标**：小说创作 44 项全部落地；30 万字连载压测通过。

**核心任务**：
- 7 层上下文装配 + 分层摘要
- 伏笔/时间线/人物弧线/防崩
- 4 家模型路由 + 降级
- 工作流 v1 可视化编排
- 自动连载 + schedules 定时
- Editor 操作集全集 + ⌘K
- Prompt 管理页 + 成本面板

## M3 内容工作室（P2）

短篇/自媒体/热点/复用/短视频/知识库全部落地。

## M4 发布·数据·出海（P3-A）

发布网关/数据回流/ROI/出海翻译管线。

## M5 协作·离线·多端（P3-B）

三角色协作/PWA 离线三级/移动端适配。

## 红线

- 任何跨里程碑抢跑功能一律拒绝
- 里程碑门禁是唯一防烂尾机制
