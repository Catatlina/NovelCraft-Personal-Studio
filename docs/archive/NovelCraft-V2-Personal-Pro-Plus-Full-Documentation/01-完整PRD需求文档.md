# 完整 PRD 需求文档 — NovelCraft Personal Studio

> 版本：V2.0 ｜ 产品定位：个人 + 1~5 人团队 AI 内容创作与运营工作台

## 1. 产品愿景与使命

**愿景**：打造一个个人或小团队专属的 AI 内容创作与运营工作台，让一个人也能拥有专业内容团队的生产效率。

**使命**：通过 AI，把创意快速转化为高质量内容，形成"选题 → 创作 → 审核 → 发布 → 数据反馈 → 持续优化"的完整闭环。

## 2. 目标用户

| 用户画像 | 核心诉求 |
|---|---|
| 独立创作者 | 用最少精力产出高质量、可持续更新的内容 |
| 网文连载作者 | 百万字设定不崩、稳定日更 |
| 自媒体矩阵运营者 | 一稿多平台分发、追热点 |
| 小团队协作成员 | 2~5 人分工协作、修改留痕 |
| 出海作者 | 中文作品翻译分发海外平台 |

## 4. 非功能需求

- **性能**：单 VPS 4C8G 支撑 1~5 人日常使用，API 响应 < 500ms
- **安全**：JWT + Fernet 加密 + CSRF + XSS 防护 + SQL 注入防护
- **可用性**：离线优先 L1~L3，PWA 支持
- **可维护性**：Docker Compose 一键部署，每日自动备份，Telegram 告警
- **可扩展性**：新的内容类型 = 新增 type + meta schema + prompt 模板，零表结构变更
