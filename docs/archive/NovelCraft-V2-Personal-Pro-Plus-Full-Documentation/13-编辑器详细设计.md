# 编辑器详细设计

## 概述

编辑器是用户停留时间最长的界面，必须达到 Notion/Cursor 级别的流畅体验。

技术选型：**Tiptap**（基于 ProseMirror），输出格式为 Tiptap JSON。

## 三栏工作台布局

```
┌──────────┬─────────────────────┬──────────┐
│  左侧栏   │      主编辑区        │  右侧栏   │
│          │                     │          │
│ · 目录   │   Tiptap 编辑器     │ · AI面板 │
│ · 人物   │   (块编辑/斜杠命令) │ · 审核   │
│ · 设定   │                     │ · 版本   │
│ · 素材   │                     │ · 大纲   │
│          │                     │          │
└──────────┴─────────────────────┴──────────┘
```

## 核心功能

### AI 浮条
选中文字后自动弹出：
```
┌─────────────────────────────────┐
│ ✨ 续写  ✏️ 改写  💎 润色       │
│ 📝 扩写  📄 缩写  🎨 去AI味   │
└─────────────────────────────────┘
```

### 斜杠命令（/）
输入 `/` 弹出命令菜单：
- `/ai` — 唤起 AI 补全
- `/continue` — AI 续写
- `/character` — 插入人物卡
- `/worldview` — 插入世界观引用
- `/summary` — 插入前情提要

### 版本侧栏
- 版本树可视化（分支/合并）
- Diff 对比视图（高亮新增/删除/修改）
- 一键恢复任意版本
- auto_save 版本（7 天滚动）/ manual 版本（永久）/ ai_rewrite 版本（永久）

### 自动保存
- 内容变更 1s 后自动保存
- 保存前自动创建版本快照
- 网络断开时暂存 IndexedDB
- 恢复网络自动提交，冲突时弹三方对比 UI

## 扩展点

Tiptap 扩展体系支持：
- Custom nodes：CharacterCard, WorldviewRef, ForeshadowFlag
- Custom marks：AIHighlight, ReviewComment
- Custom plugins：AutoVersion, OfflineQueue, WordCount

## 性能目标

- 单章 ≤ 10,000 字：渲染 < 100ms，编辑延迟 < 16ms
- 自动保存不阻塞输入
- 大文档（> 50,000 字）分页加载
