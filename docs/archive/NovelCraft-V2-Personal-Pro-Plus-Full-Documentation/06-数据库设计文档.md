# 数据库设计文档

## 概述

PostgreSQL 16 + pgvector 为唯一事实源。Redis 仅作缓存/队列/限流，数据可随时丢弃重建。

## 核心表（26 张）

### 内容与版本
- **contents**：统一内容模型，type 枚举承载所有体裁（novel/volume/chapter/short_story/wechat_article/toutiao/xhs_note/zhihu_answer/video_script...），body 为 Tiptap JSON，meta JSONB 存类型专属字段
- **derivations**：内容复用血缘（source_content_id → derived_content_id）
- **versions**：多态版本表（entity_type + entity_id + snapshot JSONB + parent_version_id 形成版本树）

### 工作流
- **workflows**：工作流定义（is_preset, definition JSONB）
- **workflow_runs**：运行实例（workflow_id, project_id, status, context JSONB）
- **run_nodes**：节点执行记录（agent, status, input/output JSONB, ai_call_ids）
- **schedules**：定时调度（workflow_id, cron, enabled）

### AI
- **ai_calls**：全量 AI 调用追踪（run_node_id, agent, task_type, provider, model, prompt_name, prompt_version, input/output JSONB, prompt_tokens, completion_tokens, cost, latency_ms, status）
- **prompts**：Prompt 库（name, version, model, template, output_schema, golden_cases）
- **model_routes**：模型路由（task_type → provider/model/params, fallbacks）
- **budgets**：预算记录（project_id, scope, limit, used）

### 知识
- **knowledge_items**：统一知识库（scope, kind, title, content, meta JSONB, source_url）
- **knowledge_vectors**：向量索引（item_id, chunk_no, embedding vector, chunk_text）

### 叙事
- **entity_states**：实体状态表（人物位置/关系/持有物）
- **foreshadowings**：伏笔（埋设章/内容/计划回收章/状态）
- **timeline_events**：时间线事件
- **arcs**：人物弧线

### 发布
- **platform_accounts**：平台账号（Fernet 加密）
- **publish_records**：发布记录
- **metrics**：数据回流（阅读/点赞/收藏/评论/分享/收益）

### 系统
- **users** / **projects** / **project_members**：用户与协作
- **sensitive_words**：敏感词表
- **audit_logs**：审计日志

## 建表拓扑顺序

```
L0  users
L1  projects
L2  project_members, platform_accounts
L3  contents, knowledge_items
L4  derivations, versions, knowledge_vectors, prompts, model_routes, workflows
L5  schedules, sensitive_words, entity_states, foreshadowings, timeline_events, arcs, reviews
L6  workflow_runs
L7  run_nodes
L8  ai_calls, budgets
L9  publish_records, metrics, audit_logs
```

## 命名规范

- 表名：复数 snake_case（users, contents, workflow_runs）
- 外键：{关系}_id（project_id, parent_id）
- 布尔：is_ 前缀（is_preset, is_deleted）
- JSONB：语义名（meta, definition, context, input, output, snapshot）
- 索引：{表}_{列}_{类型}（contents_type_idx, contents_parent_id_idx）
