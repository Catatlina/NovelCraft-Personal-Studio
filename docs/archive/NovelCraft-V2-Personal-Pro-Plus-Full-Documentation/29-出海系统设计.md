# 出海系统设计

## 概述

将中文作品翻译分发到海外平台，覆盖翻译、本地化、禁忌检查和平台适配的全链路。

## 出海全链

```
中文原文 → 分段翻译 → 文学化改写 → 文化本地化 → 禁忌检查 → SEO优化 → 海外平台发布
```

## 翻译管线

### 1. 分段翻译
- 按章节/段落拆分，保持上下文连贯
- 术语一致性：人名/地名/专有名词统一翻译
- 支持 Claude/GPT/DeepSeek 多模型

### 2. 文学化改写
- 直译 → 文学化：增强可读性、画面感、文化适配
- 对话自然化：中文"道友"→英文语境下的等价表达
- 幽默/讽刺/双关：寻找目标语言对应表达

### 3. 文化本地化
- **人名地名适配**：张三 → John / 保持拼音 + 注释
- **计量单位适配**：里 → mile / 斤 → pound
- **文化概念解释**：丹田 → dantian (energy core)
- **梗/俗语替换**：寻找目标文化等价表达

### 4. 禁忌检查
- 政治敏感内容
- 文化禁忌（宗教/种族/性别）
- 平台内容政策合规

### 5. SEO 优化
- 英文标题优化
- 关键词/标签/分类
- 简介/blurb 优化
- 目标平台算法适配

## 海外平台适配

| 平台 | 格式 | 特点 |
|---|---|---|
| **Royal Road** | Web 小说 | 章节发布/读者评论/评分系统 |
| **WebNovel** | 网文 | 章节付费/翻译质量要求高 |
| **Scribble Hub** | 网文 | 标签系统/读者互动 |
| **Amazon KDP** | 电子书/实体书 | Kindle 格式/定价/营销 |
| **Medium** | 长文 | SEO/clap/付费会员 |
| **Substack** | Newsletter | 邮件订阅/付费订阅 |

## KDP 特别适配

- EPUB/MOBI 格式生成
- 封面设计建议
- 定价策略分析
- 分类/关键词优化
- A+ 内容生成
