# SaaS 商业化设计文档 (Enterprise Edition)

> NovelCraft 当前定位为自托管工具。SaaS 商业化是**远期方向**，本文档记录可能的商业化路径以供未来参考。

## 商业化触发条件

1. ROI 面板持续正收益
2. 出现真实外部用户需求
3. 功能稳定、运维成熟
4. 有明确 SaaS 化路径

## 可能的商业化方案

### SaaS 托管版
- 用户无需部署 VPS
- 按月/年订阅
- 套餐分级：
  - **Free**：基础功能，限量
  - **Pro**：全功能，适合个人创作者
  - **Team**：协作功能，适合 3~10 人团队

### 企业版
- 10+ 人团队
- 高级协作（审批流/角色定制）
- 专属模型调优
- SLA 保障
- SSO/SCIM

### API 服务
- 叙事一致性引擎 API
- 一稿多平台 API
- 出海翻译 API

## 架构兼容性

V2.1 架构已为 SaaS 化预留空间：
- projects 表天然隔离
- project_members 支持多用户
- budgets 支持项目级预算控制
- UUIDv7 主键分布式友好

## 明确不做

- ❌ 微服务/K8s
- ❌ 复杂多租户隔离
- ❌ 自建支付系统
- ❌ 企业组织架构
- ❌ 内容内广告
