# 小说智能引擎设计文档 (Enterprise Edition)

## 概述

AI 驱动的小说创作全链路引擎，覆盖从大纲到正文、从质量保障到防崩的完整体系。

## 核心子系统

### 大纲生成
- 百万字总纲（全书结构/核心冲突/主线支线）
- 分卷大纲（卷目标/弧线阶段/伏笔计划）
- 章节细纲（场景级写作指导）

### 正文生成
- 7 层上下文装配（Token 预算制，优先级丢弃）
- 按细纲 + 上下文逐章生成
- 自动续写（章节间无缝衔接）

### 防崩体系
- 章纲先审（便宜 20 倍）→ 正文生成
- 交叉模型审（生成 vs 审核用不同 Provider）
- 每 10 章巡检 + 卷级复盘门禁

### 伏笔系统
埋设 → 追踪 → 到期强制注入 → Reviewer 校验回收

### 时间线与弧线
- 章内事件抽取入表 → 跨章矛盾检测
- 人物弧线阶段目标 → 按卷校验偏移

### 质量审核（7 维）
文笔 / 剧情 / 人物 / 设定 / 逻辑 / 伏笔 / 商业

### 编辑器操作集
续写 / 改写 / 润色 / 扩写 / 缩写 / 去 AI 味
选中浮条 / 右键菜单 / ⌘K 全局命令

## 技术指标
- 单书 ≥ 100 万字
- 30 万字压测通过
- 一致性错误率 < 5%
- 7 维均分 ≥ 80
