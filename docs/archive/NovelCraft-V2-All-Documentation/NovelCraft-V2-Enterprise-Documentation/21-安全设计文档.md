# 安全设计文档 (Enterprise Edition)

## 认证授权

- JWT（access 15min + refresh 7day）
- RBAC 三角色（Owner/Editor/Viewer）
- API 层装饰器校验
- Repository 基类强制归属过滤

## 数据加密

- API Key / 平台账号：Fernet 对称加密
- JWT Secret：环境变量
- 备份：age 加密 + 异地传输

## Web 安全

| 威胁 | 防护 |
|---|---|
| CSRF | SameSite Cookie + Token |
| XSS | Tiptap schema 白名单 + 输出转义 |
| SQL 注入 | ORM 参数化 + CI 静态检查 |
| 信息泄露 | 错误响应脱敏 |

## AI 安全

- Prompt 注入防护（变量注入，非字符串拼接）
- 内容安全过滤（敏感词 + 平台规则）
- 仿写防侵权（相似度三道闸）
- 预算熔断（防恶意消耗）
- 速率限制（令牌桶）

## 审计

audit_logs 中间件级记录：
谁/何时/对什么/做了什么/IP/UA

关键操作（删除/权限变更/预算修改/全自动发布开启）强制写 audit_logs。

## 上线检查清单

- [ ] JWT Secret ≥ 256 bit
- [ ] API Key 已加密
- [ ] HTTPS 已配置
- [ ] 数据库密码非默认
- [ ] Redis 密码已设置
- [ ] CORS 白名单限定
- [ ] 备份恢复已测试
- [ ] 告警通道已验证
