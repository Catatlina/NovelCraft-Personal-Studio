# 自动连载生产系统设计 (Enterprise Edition)

## 概述

全自动的小说连载生产系统，从大纲到发布的全链路无人值守。

## 自动连载工作流

```
定时触发(beat) → 检查待续写小说 → 生成细纲 → 细纲审核(human,可选)
  → 生成正文 → 7维审核 → 质量分<80? → 自动返工(≤2轮)
  → 自动润色 → 可选自动发布 → 继续下一章
```

## 定时策略

- **日更模式**：每天固定时间生成 1~2 章
- **批量模式**：一次性生成 N 章后暂停
- **追赶模式**：检测到存量不足时加速生成
- **智能调速**：根据质量分动态调整生成速度

## 防崩机制

- 每 10 章自动巡检（全局一致性扫描）
- 每卷完成强制 human 复盘门禁
- 质量分连续 3 章下降 → 暂停 + 告警
- 月度全量一致性审计

## 断点续跑

- 系统重启后从上次成功章节继续
- 生成失败自动重试（≤ 2 次）
- 超预算自动暂停（PENDING_BUDGET）
- 所有状态可恢复

## 监控面板

- 当前连载进度（章数/字数/质量趋势）
- 伏笔回收率
- 成本统计（每章/累计/预估）
- 异常告警（质量下降/成本异常/Provider 故障）
