# 前端工程规范 (Enterprise Edition)

## 技术栈

- React 18 + TypeScript strict
- Vite 构建
- Radix UI + shadcn/ui + Tailwind CSS
- Tiptap 编辑器
- Zustand + TanStack Query
- PWA (vite-plugin-pwa)

## 目录结构

```
frontend/src/
├── design/          # tokens.css · primitives · motion
├── components/      # editor/ workflow/ novel/ studio/ common/
├── pages/           # 路由页面
├── stores/          # zustand stores
├── offline/         # Service Worker · IndexedDB · outbox
├── api/             # openapi-ts 客户端
└── hooks/           # 自定义 hooks
```

## 设计规范

- 所有视觉决策来自 Design Tokens
- 禁止在业务代码写裸色值/裸间距/裸字号
- 暗色模式原生（data-theme="dark" 默认）
- 对标 Linear/Notion/Cursor/Raycast

## 组件规范

- 每个组件有 TypeScript 接口 + Storybook
- 空状态/加载态/错误态/边界情况全部处理
- 优先用 Radix 无头组件，不重复造轮子

## 性能

- 路由级代码分割
- 大列表虚拟滚动
- 编辑器增量同步（debounce 1s）
- SSE 连接复用

## 代码质量

- ESLint + Prettier 自动格式化
- TypeScript strict mode
- PR 必须通过 lint + type-check + build
