# 版本规划路线图 (Enterprise Edition)

## 版本策略

版本号：`v{major}.{minor}.{patch}-{milestone}`

- **major**：架构级变更
- **minor**：里程碑交付
- **patch**：Bug 修复/小优化
- **milestone**：M1~M5

## 里程碑总览

| 版本 | 里程碑 | 周期 | 交付 |
|---|---|---|---|
| v2.0.0-m1 | M1 地基+MVP | 第 1~5 周 | 灵感→第一章闭环 |
| v2.0.0-m2 | M2 小说引擎 | 第 6~16 周 | 30 万字防崩连载 |
| v2.1.0-m3 | M3 内容工作室 | 第 17~26 周 | 全体裁+知识库+热点 |
| v2.2.0-m4 | M4 发布出海 | 第 27~34 周 | 发布+回流+ROI |
| v2.3.0-m5 | M5 协作离线 | 第 35~40 周 | 小团队+离线+移动 |
