# NovelCraft V2.0 详细PRD产品需求文档

## 产品目标

打造面向个人作者、小说工作室、内容公司的AI小说生产SaaS。

## 用户角色

### 普通作者

-   创建小说
-   AI辅助创作
-   管理章节

### 专业作者

-   多项目管理
-   数据分析
-   自动连载

### 工作室

-   团队协作
-   账号管理
-   批量生产

# 核心页面

## Dashboard

功能： - 小说项目列表 - 生产状态 - AI任务状态 - 数据统计

## 小说工作台

包含： - 总纲管理 - 世界观管理 - 人物管理 - 章节管理 - 编辑器 - AI助手

## AI生产中心

流程：

灵感输入 → 市场分析 → 大纲生成 → 章节生产 → 自动审核 → 发布

# UI要求

参考： - 番茄作家助手 - Notion - Sudowrite

要求： - PC优先 - 移动端适配 - 暗色模式 - 高级编辑体验

# 商业化

套餐： - 免费版 - 专业版 - 工作室版

计费： - Token - AI调用次数 - 项目数量
