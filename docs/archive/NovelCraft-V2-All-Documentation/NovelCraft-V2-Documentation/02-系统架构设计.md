# NovelCraft V2.0 系统架构设计

## 总体架构

Frontend ↓ API Gateway ↓ Business Core ↓ AI Orchestration ↓ Model Router
↓ Data Layer

## 技术栈

前端： - React - TypeScript - Vite - Tailwind - Tiptap

后端： - FastAPI - PostgreSQL - Redis - Celery - pgvector

## AI架构

Agent Manager统一管理多个专业Agent。

支持： - GPT - Claude - DeepSeek - Gemini

## 部署

Docker Compose部署。 支持开发环境、测试环境、生产环境分离。
