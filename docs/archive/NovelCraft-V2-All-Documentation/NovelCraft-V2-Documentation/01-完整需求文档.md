# NovelCraft V2.0 完整需求文档

## 产品定位

AI驱动的全自动小说工业化生产系统。

目标：覆盖选题、规划、创作、审核、发布、数据反馈完整链路。

## 核心能力

-   AI小说生产流水线
-   多Agent协作
-   百万字长篇规划
-   自动续写
-   剧情一致性控制
-   质量审核
-   出海发布

## AI生产流程

灵感分析 → 市场分析 → 爆款拆解 → 商业定位 → 世界观生成 → 人物设计 → 总纲
→ 卷纲 → 章节规划 → 正文生成 → 审核修改 → 发布。

1.  Producer Agent
2.  Story Architect Agent
3.  Character Agent
4.  Outline Agent
5.  Writer Agent
6.  Continuity Agent
7.  Style Agent
8.  Editor Agent
9.  Market Agent
10. Publisher Agent

## 平台目标

PC Web、Mobile Web、Android、iOS。 UI对标商业写作软件。
