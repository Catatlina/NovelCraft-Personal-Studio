# NovelCraft V2.0 UI设计规范

## 设计目标

打造商业级AI写作平台。

参考：

-   番茄作家助手
-   Notion
-   Sudowrite

## 整体风格

要求：

-   简洁
-   专业
-   高信息密度
-   暗色模式支持

## 页面结构

Dashboard

包含：

-   小说卡片
-   生产状态
-   AI任务
-   数据指标

## 小说工作台

三栏布局：

左侧： - 目录 - 世界观 - 人物

中间： - 编辑器

右侧： - AI助手 - 审核结果 - 修改建议

## 编辑器功能

支持：

-   富文本
-   AI续写
-   AI改写
-   选中优化
-   历史版本

## 移动端

支持：

-   手机阅读
-   快速修改
-   AI对话

## 组件规范

统一：

-   Button
-   Modal
-   Card
-   Editor
-   Timeline
-   Dashboard Widget

## 交互要求

所有AI任务：

显示：

-   当前阶段
-   进度
-   Token消耗
-   预计完成时间
