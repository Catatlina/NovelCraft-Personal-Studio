# NovelCraft Personal Studio 开发路线图（全功能覆盖版）

> 原则变化：不再砍功能，改为**全功能分期覆盖**。每个里程碑结束系统必须"日常可用且自洽"。周期按每周 15~25 小时估算；里程碑门禁：M(n) 未验收不开 M(n+1)。清单 250+ 功能点在 §6 给出里程碑映射，保证"一个不少"。

---

## P0 / M1 — 地基 + MVP 闭环（第 1~5 周）

**目标**：V1 收口至 80 分；设计系统建成；C1/C2/C3/C5/C8 五个平台能力以最终表结构落地；跑通"注册→创建小说→灵感→六件套→审核→编辑→保存"。

**功能**：V1 遗留收口（slowapi 定型、Celery 重试/死信、备份+告警）；设计系统 v1（tokens/组件库/暗色）；统一 Content 模型 + V1 数据迁移；通用版本系统（自动保存快照/恢复）；AI Gateway v1（DeepSeek + ai_calls 全量追踪 + 三级预算）；工作流引擎 v0（chain 级：六件套 bootstrap 即第一条预设工作流，含 human 确认节点）；7 维审核接入；Tiptap 编辑器基础版（编辑/自动保存/版本恢复/选中润色·改写·续写三件套）。

**DB**：contents、derivations、versions、workflows、workflow_runs、run_nodes、ai_calls、budgets、prompts、model_routes、project_members、audit_logs（一次性建齐骨架表）。
**API**：/contents CRUD、/novels/{id}/bootstrap、/runs/{id}(+SSE)、/versions、节点重试。
**前端**：创建向导、生成进度页、六件套审阅页（雷达图）、编辑器页。

**验收**：15 分钟灵感→过审第一章；杀 worker 断点续跑；预算熔断生效；版本恢复正确；ai_calls 记录完整可查；单测 ≥135 全绿；备份恢复演练通过。

---

## P1 / M2 — 小说引擎完全体（第 6~16 周）★护城河

**目标**：清单"小说创作"44 项全部落地；30 万字连载压测通过。

**功能**：分卷大纲/章节细纲/黄金三章工作流；自动续写与自动连载（定时 schedules）；7 层上下文装配；分层摘要；实体状态表；伏笔种植/追踪/回收 + 看板；时间线管理；人物弧线；OOC/设定冲突/前文一致性/节奏检测（Reviewer 扩维）；防崩巡检（每 10 章）+ 卷级复盘门禁；质量评分+自动返工（branch 节点）+ 多轮审核；交叉模型审核（接入 Claude/OpenAI/Gemini Provider，模型路由/降级完整版）；Editor 操作集（去AI味/润色/扩写/缩写/改写）编辑器全接入；工作流引擎 v1（可视化编排：节点开关/排序/模型与 Prompt 选择/暂停恢复取消）；Prompt 库管理页 + golden case 回归；多书管理/多项目并行；⌘K 命令面板；成本面板（单章/单篇/分模型）。

**验收**：一本书连续生成 30 万字，人工抽检一致性错误率 <5%，7 维均分 ≥80；伏笔到期回收提醒与校验生效；自定义工作流可从零编排一条"细纲→正文→审核→返工"链并成功执行；4 家模型路由与降级演练通过。

---

## P2 / M3 — 内容工作室（第 17~26 周）

**目标**：清单"短篇/自媒体/热点/复用/短视频/素材知识"六类全部落地。

**功能**：短篇体系（短篇/微小说/情感/悬疑/爆款结构模板）；自媒体十平台内容类型（公众号/头条/小红书/百家号/知乎/大鱼/网易/Medium/Substack/X）+ 平台风格适配 + 爆款/SEO 标题、摘要导语、标签关键词、封面文案、配图建议、评论引导、CTA（= 各 type 的 meta 生成字段）；一稿多平台 fan-out 工作流 + 批量生成 + 内容矩阵管理页；短视频脚本族（抖音/小红书/B站：3 秒钩子/分镜/旁白/口播/镜头建议）；Knowledge Hub 完整版（素材中心：文本/图片/PDF/Word/MD/链接入库解析，金句库/标题库/平台规则库/品牌风格库，项目级+全局，向量检索页）；风格学习 + 仿写辅助（含相似度防侵权）+ 拆书分析工作流；热点系统（可插拔采集 adapter：新闻热榜/行业热点/AI·科技·财经·商业专题 + 价值评估/平台适配判断/选题推荐/选题库/趋势·爆款·扫榜·开篇·爽点·节奏·标签·传播潜力分析 Agent）+ 每日热点晨报定时工作流；Prompt 实验室 + A/B 测试 + 多模型对比视图。

**验收**：一章小说 5 分钟派生 5 平台格式且血缘可查；每日晨报工作流连续 7 天无人值守成功（human 节点等待审核）；知识库对 100+ 素材检索相关性抽检通过；仿写相似度红线拦截生效。

---

## P3-A / M4 — 发布·数据·出海（第 27~34 周）

**功能**：发布网关（账号管理/草稿/定时/发布记录/失败重试/格式适配）；三模式发布（手动/半自动默认/官方 API 平台全自动：Medium、Substack、X、WordPress；国内自动化显式开启+风险确认）；敏感词与平台规则前置过滤；数据回流（API 自动 + 半自动录入 + 手动兜底：阅读/点赞/收藏/评论/分享/收益）；内容表现分析 + 平台对比 + 内容优化建议 Agent + ROI 面板；出海全链（分段翻译/文学化改写/文化本地化/人名地名与计量适配/禁忌检查/SEO —— 复用 V1 三段式管线重构为工作流）+ Royal Road/WebNovel/Scribble Hub/Amazon KDP 格式适配与出海发布记录。

**验收**：一篇内容从生成到（半自动）发布到数据回流全链可查；出海工作流产出一章英文稿并通过禁忌检查；ROI 面板展示成本-收益对照。

---

## P3-B / M5 — 协作·离线·多端（第 35~40 周）

**功能**：协作（Owner/Editor/Viewer、项目共享、操作日志页、修改记录视图）；Offline First L1→L2→L3（PWA、只读缓存、离线编辑暂存、AI 出站队列、冲突版本树对比 UI）；移动端 Web 适配（响应式工作台精简版）；Agent 状态追踪控制台（run_nodes 可视化）。

**验收**：第二个账号以 Editor 加入协作编辑无越权；断网状态可浏览/搜索/编辑并排队 AI 任务，恢复后自动执行；手机浏览器可完成"审核确认 + 轻量编辑"。

**后续（不排期）**：Android/iOS 原生、更多平台 adapter、SaaS 化预研（触发条件：ROI 面板持续正收益 + 出现真实外部用户需求）。
